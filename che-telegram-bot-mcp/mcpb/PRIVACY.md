# Privacy Policy

## Data Collection

This MCP server:
- Runs entirely locally on your Mac
- Does not collect or transmit any personal data to third parties
- Communicates only with the Telegram Bot API (api.telegram.org) using your provided bot token
- Does not store messages or chat data locally

## Credentials

- Your Telegram Bot Token is read from the `TELEGRAM_BOT_TOKEN` environment variable
- The token is used only for authenticating with the Telegram Bot API
- The token is never logged or stored beyond process memory

## Third-Party Services

This extension communicates with:
- **Telegram Bot API** (https://api.telegram.org) - to execute bot operations
